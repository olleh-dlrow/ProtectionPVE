# 精简版第三人称射击局域网游戏

## 简介

游戏演示及可执行文件：

https://pan.baidu.com/s/1dHuRHf0wgYaQ_soSjMhN8g 提取码：2048

## 启动方法

1. 运行NetworkTest/WindowServer/Server.bat
1. 运行NetworkTest/WindowsNoEditor/ProtectionPVE.exe
1. 重复步骤2

## 操作说明

移动：WSAD
开火：鼠标左键
更换武器：1, 2
拾起武器：E
投掷手雷：T
跳跃：Space
蹲下：C
换弹：R
退出对局：Esc

## 游戏规则

玩家在限定时间内保护NPC到达目标地点
